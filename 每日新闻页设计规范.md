# 每日新闻页设计规范

> 基于 `/workspace/ai-news-daily/pages/daily-news.html` 源文件及 `/workspace/ai-news-daily/colors_and_type.css` 设计系统Token文件逐项提取，所有数值均来自实际代码。

---

## 1. 页面概述

### 1.1 页面用途

"每日新闻"页是 AI日报 网站的核心内容展示页面，用于呈现当日AI领域全部精选新闻条目。每张卡片代表一条独立新闻，包含标题、分类标签、发布时间和正文摘要。

### 1.2 目标受众

- AI领域从业者、技术开发者、研究人员
- 关注AI行业动态的投资人与决策者
- 对人工智能新闻感兴趣的普通读者

### 1.3 页面在站点流程中的角色

- **首页（index.html）**：站点入口，展示今日摘要卡片 + 历史新闻时间线 + 关于板块
- **每日新闻页（daily-news.html）**：首页"查看今日详情"的详情页，完整展示当日所有新闻条目
- 页面顶部导航栏提供"回到首页"链接，构成二级页面导航关系

### 1.4 内容结构概览

页面自上而下依次为：

1. 固定顶部导航栏（反转样式，黑色背景）
2. 大号日期头部（日期 + 星期副标题）
3. 8张新闻卡片（垂直排列，分类色左边框 + 图标）
4. 页脚（免责声明）
5. 回到顶部按钮（固定定位，滚动后出现）

## 2. 设计风格定义

### 2.1 视觉风格

**Brutalist News Feed（粗野主义新闻流）**

整体视觉语言延续了首页的 Brutalist 设计体系：全等宽字体、零圆角、硬阴影、2px实线边框、黑白色调。但通过分类色编码系统和反转导航栏，在保持体系一致性的同时建立了子页面的视觉识别。

### 2.2 设计理念

| 原则     | 具体表现                                            |
| ------ | ----------------------------------------------- |
| 全等宽字体  | 页面全局使用 `Geist Mono` 等宽字体，营造终端/技术文档感             |
| 零圆角    | 所有圆角Token均为 `0px`，卡片、按钮、图标容器均为直角                |
| 硬阴影    | 阴影采用 `2px 2px 0 rgba(0,0,0,1)`，无模糊扩散，产生"偏移叠加"效果 |
| 分类色彩编码 | 5种新闻分类各有独立色值，通过卡片左边框和图标传递视觉分类信号                 |
| 反转导航栏  | 导航栏使用黑色背景 + 白色前景，与首页正常导航形成对比                    |
| 内联样式驱动 | 绝大多数样式通过HTML内联 `style` 属性实现，不依赖Tailwind工具类      |

### 2.3 与首页风格的差异

每日新闻页在保持Brutalist体系的基础上，有以下差异化处理：

- 导航栏背景色反转（黑色 vs 卡片色）
- 卡片采用分类色左边框 + 图标容器（首页卡片无此设计）
- 日期头部采用 3rem 超大字号（首页日期为 text-sm/text-base）
- 卡片结构包含图标容器、标题、分类时间标签、正文（首页卡片为纯文本 + 链接）
- 回到顶部按钮无hover transform效果（首页有）
- 页面底部额外包含 `box-sizing: border-box` 重置样式块
- body元素无Tailwind工具类（首页body有 `class="min-h-screen antialiased"`）

## 6. 组件规范

### 6.1 固定顶部导航栏（反转样式）

#### 6.1.1 HTML结构

```html
<nav style="position: fixed; top: 0; left: 0; right: 0; z-index: 50;
            background: var(--color-text-foreground);
            border-bottom: 2px solid var(--color-border-border);
            height: 58px;">
  <div style="max-width:1200px; margin:0 auto; padding:0 16px;
              height:100%; display:flex; align-items:center;
              justify-content:space-between;">
    <!-- 左侧：Logo + 品牌文字 -->
    <a href="#" style="display:flex; align-items:center; gap:8px;
                       text-decoration:none; color:var(--color-border-border);">
      <svg ...>...</svg>
      <span style="font-size:1rem; font-weight:700; letter-spacing:-0.02em;">AI日报</span>
    </a>
    <!-- 右侧：回到首页链接 -->
    <a href="#" data-dom-id="nav-home" data-nav-key="home"
       style="color:var(--color-border-border); text-decoration:none;
              font-size:0.875rem; border:2px solid var(--color-border-border);
              padding:4px 12px;">&#8592; 回到首页</a>
  </div>
</nav>
```

#### 6.1.2 完整CSS属性

**外层 `<nav>` 元素：**

| 属性              | 值                                      | 说明                        |
| --------------- | -------------------------------------- | ------------------------- |
| `position`      | `fixed`                                | 固定定位，不随滚动移动               |
| `top`           | `0`                                    | 紧贴视口顶部                    |
| `left`          | `0`                                    | 撑满左侧                      |
| `right`         | `0`                                    | 撑满右侧                      |

**JavaScript实现：**

```javascript
(function(){
  var b = document.getElementById('back-to-top');
  window.addEventListener('scroll', function(){
    if(window.scrollY > 300){
      b.style.opacity = '1';
      b.style.pointerEvents = 'auto';
    } else {
      b.style.opacity = '0';
      b.style.pointerEvents = 'none';
    }
  });
})();
```

## 10. 响应式适配

### 10.1 当前行为

本页面**未实现响应式适配**。所有样式均通过HTML内联 `style` 属性定义，未使用Tailwind的响应式工具类（如 `md:`、`lg:` 前缀）。具体表现：

- 导航栏高度固定 `58px`（无响应式变化，首页使用 `h-12 md:h-14` 即 48px/56px）
- 主区域内边距固定 `64px 16px 48px`（无小/中/大屏适配）
- 卡片内flex布局无换行处理（窄屏下图标+文字可能挤压）
- 日期标题 `3rem`（48px）在移动端可能过大
- 卡片间距固定 `16px`
- 页脚内边距固定 `32px 16px`
- 回到顶部按钮固定 `40px x 40px`，定位 `bottom:24px; right:24px`

### 10.2 推荐改进

| 断点             | 组件       | 建议调整                                |
| -------------- | -------- | ----------------------------------- |
| `≤640px` (移动端) | 导航栏高度    | 可减至 `48px`，并相应调整 main 的 padding-top |
| `≤640px`       | 日期标题     | `3rem` → `2rem` 或 `2.25rem`         |
| `≤640px`       | 卡片内布局    | 可改为垂直布局（图标在上，内容在下）                  |
| `≤640px`       | 主区域左右内边距 | `16px` 可保持或增至 `20px`                |
| `≤640px`       | 卡片内边距    | `16px 20px` → `12px 16px`           |

| `≤768px`       | 日期标题     | `3rem` → `2.25rem`                  |

### 14.3 JavaScript

页面仅包含一个IIFE（立即执行函数表达式），用于控制回到顶部按钮的显隐：

```javascript
(function(){
  var b = document.getElementById('back-to-top');
  window.addEventListener('scroll', function(){
    if(window.scrollY > 300){
      b.style.opacity = '1';
      b.style.pointerEvents = 'auto';
    } else {
      b.style.opacity = '0';
      b.style.pointerEvents = 'none';
    }
  });
})();
```

### 14.4 样式层级（优先级从低到高）

1. `<style type="text/tailwindcss">` — Tailwind @layer base（body背景/文字色）
2. `<style>` 通用样式块 — `.no-scrollbar`、`[data-icon]`
3. `colors_and_type.css`（内联于 `<style id="theme-vars">`） — CSS自定义属性定义
4. 页面底部 `<style>` 块 — `box-sizing: border-box`、body字体/背景
5. 内联 `style` 属性 — 所有组件的具体样式（最高优先级）
