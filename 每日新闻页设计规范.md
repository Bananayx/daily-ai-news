## 1. 页面概述

### 1.1 页面定位

"每日新闻"页是AI日报网站的核心内容页，用于每日聚合呈现当日AI领域的重要新闻。该页面是用户从首页点击日期或直接访问URL后进入的详情页面，承载全站最核心的信息消费功能。

### 1.2 目标受众

- AI行业从业者（研究员、工程师、产品经理）
- 科技媒体记者与编辑
- 关注AI领域动态的投资人及决策者
- 对人工智能技术发展感兴趣的普通用户

### 1.3 页面在站点流程中的角色

```
首页（日期列表）→ 每日新闻页（当日新闻详情）
```

用户可通过以下路径到达本页：
1. 首页点击某日日期标题进入
2. 直接访问URL路径 `/daily/2026-06-27`
3. 首页导航栏中的日期链接

本页提供"← 回到首页"的返回导航，构成完整的页面闭环。

### 1.4 页面内容结构概览

本页包含以下主要区域（自上而下）：

| 区域 | 内容 |
|------|------|
| 固定导航栏 | AI标识 + "AI日报" 品牌名 / "← 回到首页" 返回链接 |
| 日期头部 | 大字日期 "2026.06.27" + 副标题 "星期六 · 今日AI新闻速递" |
| 新闻卡片列表 | 8张新闻卡片垂直排列，每张含分类图标、标题、分类标签、时间、描述 |
| 页脚 | 版权信息 + 数据来源声明 |
| 回到顶部按钮（悬浮） | 固定在右下角的快捷滚动按钮 |

## 2. 设计风格定义

### 2.1 视觉风格

**布鲁塔主义新闻信息流（Brutalist News Feed）**

本页采用硬朗、直接、无装饰的布鲁塔主义设计风格，所有视觉元素传达原始、真实的质感，拒绝柔和阴影和圆角装饰，以高对比度的几何形状和粗线条构建信息层级。

### 2.2 设计基调

| 维度 | 描述 |
|------|------|
| 视觉感受 | 冷峻、专业、信息密度高、不加修饰 |
| 情感调性 | 理性、客观、技术感 |
| 信息传达方式 | 直截了当，结构优先于装饰 |
| 视觉层次 | 通过色彩编码（分类颜色）、字号梯度、硬阴影建立层级 |

### 2.3 核心设计哲学

1. **零圆角（Zero Radius）**：页面上所有元素（卡片、按钮、图标容器、导航栏）均不使用任何圆角，全部为直角矩形，体现布鲁塔主义的原始感。
2. **硬阴影（Hard Shadow）**：所有卡片使用 2px 2px 0 rgba(0,0,0,1) 的纯黑无模糊偏移阴影，不使用柔和的 box-shadow 模糊效果。
3. **等宽字体（Monospace Typography）**：全站使用 Geist Mono 等宽字体，强化技术感和代码化的视觉风格。
4. **分类色彩编码（Category Color Coding）**：5种新闻分类各有专属颜色，通过左侧色条、图标边框、图标描边三重方式传达分类信息，形成快速视觉识别。
5. **反转导航栏（Inverted Navbar）**：与首页白色导航栏形成对比，详情页使用黑色背景的反转导航栏，通过视觉反转暗示页面层级变化。
6. **粗白边框（Bold White Border）**：卡片外围使用 2px solid #ffffff 的白色边框，与灰色背景形成清晰分离。
7. **信息密度优先**：每张卡片在紧凑空间内包含完整信息（图标、标题、分类、时间、描述），不做牺牲信息量的简化。

### 2.4 设计语言关键词

```
RAW  ·  DIRECT  ·  MONOSPACE  ·  HIGH-CONTRAST  ·  ZERO-DECORATION  ·  STRUCTURED  ·  BRUTALIST
```

## 3. 颜色系统

### 3.1 基础颜色令牌

| 令牌名称 | 色值 | CSS 变量 | 用途 |
|----------|------|----------|------|
| 页面背景 | `#c5c9c9` | `--color-bg` | 页面整体背景色（body） |
| 卡片背景 | `#d8dada` | `--color-card-bg` | 新闻卡片背景色 |
| 导航栏背景 | `#111111` | `--color-navbar-bg` | 固定顶部导航栏背景（反转色） |
| 主文字色 | `#111111` | `--color-text` | 标题、正文、主要可读文字 |
| 次文字色 | `#888888` | `--color-text-muted` | 副标题、时间标签、页脚、辅助信息 |
| 白色边框 | `#ffffff` | `--color-border` | 卡片外边框（2px solid） |
| 纯白 | `#ffffff` | `--color-white` | 导航栏文字、按钮文字、图标填充 |
| 纯黑 | `#111111` | `--color-black` | 导航栏背景、回到顶部按钮背景 |
| 阴影色 | `rgba(0,0,0,1)` | `--color-shadow` | 卡片硬阴影颜色（完全不透明纯黑） |

### 3.2 新闻分类色彩系统

每类新闻拥有一个专属主题色，用于左侧色条、图标容器边框、图标描边三处。

| 分类名称 | 色值 | CSS 变量 | 语义含义 | 页面使用次数 |
|----------|------|----------|----------|-------------|
| 研究突破 | `#0040ff` | `--color-category-research` | AI前沿学术研究、论文发表、理论突破 | 2次 |
| 产品发布 | `#1a8a4a` | `--color-category-product` | 新产品上线、功能更新、商业化发布 | 2次 |
| 行业动态 | `#b8860b` | `--color-category-industry` | 企业动态、市场趋势、行业事件 | 2次 |
| 政策法规 | `#d73333` | `--color-category-policy` | 法律法规、政策变动、监管动态 | 1次 |
| 开源项目 | `#6b90ff` | `--color-category-opensource` | 开源代码发布、社区项目、工具开源 | 1次 |

### 3.3 分类色彩使用规则

每个分类颜色在同一张卡片内以三种方式出现：

1. **左侧色条**：`border-left: 4px solid {分类色}`
2. **图标容器边框**：`border: 2px solid {分类色}`
3. **图标描边色**：`stroke: {分类色}`

> 严格规则：一个分类的颜色**只**用于该分类对应的卡片，不得跨分类混用。分类色不用于文字或背景，仅用于边框和图标描边。

### 3.4 色彩层级总览

```
层级 0（最高对比）：  #111111 文字 on #ffffff 背景  →  导航栏内
层级 1（高对比）：    #111111 文字 on #d8dada 背景  →  卡片内文字
层级 2（中对比）：    #111111 文字 on #c5c9c9 背景  →  日期头部、页脚
层级 3（低对比）：    #888888 文字 on #d8dada 背景  →  卡片内辅助信息
层级 4（装饰色）：    分类色 作为边框/描边           →  分类标识
层级 5（硬阴影）：    rgba(0,0,0,1) 偏移            →  卡片立体感
```

## 4. 字体排版

### 4.1 字体族

```css
font-family: 'Geist Mono', ui-monospace, SFMono-Regular, 'SF Mono', Menlo, Consolas, 'Liberation Mono', monospace;
```

- **首选字体**：Geist Mono（Vercel 出品的等宽字体）
- **回退字体链**：ui-monospace → SFMono-Regular → SF Mono → Menlo → Consolas → Liberation Mono → monospace
- **全局强制**：页面中所有文本元素（包括导航、标题、正文、标签、页脚）均使用此字体族，无例外。

### 4.2 字号梯度表

| 元素 | 字号 | 字号（px） | 字重 | 行高 | 字间距 | 颜色 | 其他 |
|------|------|-----------|------|------|--------|------|------|
| 导航栏品牌名 "AI日报" | 1.125rem | 18px | 700 (bold) | — | — | #ffffff | — |
| 日期头部主日期 "2026.06.27" | 3rem | 48px | 700 (bold) | 1.1 | -0.03em | #111111 | 大字号、窄间距营造压迫感 |
| 日期头部副标题 "星期六 · 今日AI新闻速递" | 1rem | 16px | 400 (normal) | 1.5 | 0 | #888888 | muted色 |
| 新闻卡片标题 | 1.125rem | 18px | 700 (bold) | 1.4 | 0 | #111111 | — |
| 新闻卡片分类+时间标签 | 0.75rem | 12px | 400 (normal) | 1 | 0 | #888888 | muted色 |
| 新闻卡片描述正文 | 0.875rem | 14px | 400 (normal) | 1.6 | 0 | #111111 | 较松行高提升可读性 |
| 导航栏返回链接文字 | 0.875rem | 14px | 400 (normal) | — | 0 | #ffffff | — |
| 页脚版权信息 | 0.875rem | 14px | 400 (normal) | 1.5 | 0 | #888888 | muted色 |
| 页脚来源声明 | 0.75rem | 12px | 400 (normal) | 1.5 | 0 | #888888 | muted色 |

### 4.3 字重规范

| 字重值 | 用途 |
|--------|------|
| 700 (bold) | 日期头部、卡片标题、导航栏品牌名 |
| 400 (normal) | 其余所有文本（描述、标签、页脚、副标题、链接） |

### 4.4 文字装饰

- **链接**：无下划线，无颜色变化（在导航栏中保持白色文字），悬停时无额外装饰
- **标题**：无下划线，无装饰线
- 所有文字无 `text-decoration`、无 `text-transform`

## 5. 间距系统

### 5.1 基准单位

**基准间距单位：4px**

所有间距值为 4px 的整数倍，遵循 4px 基线网格。

### 5.2 间距令牌表

| 间距名称 | 值 | 用途 |
|----------|-----|------|
| xs | 4px | 导航栏内边距（垂直）、返回按钮内边距（垂直） |
| sm | 8px | 图标容器与内容区之间的额外微调 |
| md | 12px | 图标容器与文字内容之间的水平间距（gap） |
| base | 16px | 页面容器左右内边距、卡片内边距、卡片之间垂直间距、导航栏内边距 |
| lg | 20px | 卡片内容区域的垂直内边距（padding-top/bottom） |
| xl | 24px | 页脚上内边距（padding-top） |
| 2xl | 32px | 日期头部底部外边距（margin-bottom） |
| 3xl | 48px | 页脚上外边距（margin-top）、主内容区顶部内边距（padding-top，为固定导航栏留空） |

### 5.3 间距详细说明

**页面容器（Container）**

```css
max-width: 1200px;
padding-left: 16px;
padding-right: 16px;
margin-left: auto;
margin-right: auto;
```

**固定导航栏（Navbar）**

```css
padding: 16px;              /* 四周均为16px */
height: auto;               /* 由内容决定，非固定高度 */
```

**主内容区（Main Content）**

```css
padding-top: 80px;          /* 为固定导航栏留出空间 */
padding-bottom: 0;          /* 无底部额外间距 */
```

**日期头部（Date Header）**

```css
margin-bottom: 32px;        /* 与第一张卡片之间的间距 */
```

**新闻卡片列表（Card Stack）**

```css
display: flex;
flex-direction: column;
gap: 16px;                  /* 卡片之间间距 */
```

**单张新闻卡片（Card Internal）**

```css
padding: 20px 16px;         /* 垂直20px，水平16px */
```

**卡片内部布局间距**

```css
/* 图标容器与文字内容区之间 */
gap: 12px;
```

**页脚（Footer）**

```css
margin-top: 48px;
padding-top: 24px;
```

## 6. 组件规范

### 6.1 固定导航栏（反转样式）

#### 定位与层级

```css
position: fixed;
top: 0;
left: 0;
right: 0;
z-index: 50;
```

#### 视觉样式

```css
background-color: #111111;           /* 纯黑背景，与首页白色形成反转 */
border-bottom: 2px solid #ffffff;   /* 底部白色粗边框 */

```

#### 内部布局

```
┌──────────────────────────────────────────────────────────────────────┐
│  🤖 AI日报                                    ← 回到首页  [2px边框] │
│  ↑ 左侧品牌区                                    ↑ 右侧返回按钮    │
└──────────────────────────────────────────────────────────────────────┘
```

使用 Flexbox 水平布局：

```css
display: flex;
justify-content: space-between;
align-items: center;
max-width: 1200px;
margin: 0 auto;
padding: 16px;
```

#### 品牌区（左侧）

- **AI图标**：24x24px SVG（详见第9节SVG图标规范），白色描边
- **品牌名 "AI日报"**：
  - 字号：1.125rem（18px）
  - 字重：700（bold）
  - 颜色：#ffffff
  - 图标与文字间距：8px

#### 返回链接按钮（右侧）

```css
/* "← 回到首页" 文字链接 */
a {
  font-size: 0.875rem;             /* 14px */
  color: #ffffff;
  text-decoration: none;
  border: 2px solid #ffffff;       /* 白色边框包裹 */
  padding: 4px 12px;                /* 上下4px，左右12px */
  display: inline-flex;
  align-items: center;
  gap: 6px;                        /* 箭头与文字间距 */
}
```

- 箭头符号 "←" 为纯文本字符（Unicode U+2190），非SVG
- 悬停效果：无变色、无阴影变化，仅 `cursor: pointer` 指针变化

### 6.5 新闻卡片图标容器

#### 容器样式

```css
.icon-container {
```

#### 内部SVG图标

```css
.icon-container svg {
  width: 20px;                          /* 图标尺寸 */

  stroke: {分类色};                      /* 描边色与分类色一致 */

```

## 8. 交互规范

### 8.1 交互原则

本页遵循**极简交互**原则。布鲁塔主义风格要求交互反馈克制、直接，不使用动画过渡或视觉装饰。

### 8.2 回到顶部按钮

```javascript
// 点击行为
document.querySelector('.back-to-top').addEventListener('click', () => {
  window.scrollTo({ top: 0, behavior: 'smooth' });
});
```

- **滚动方式**：`behavior: 'smooth'`（平滑滚动）
- **滚动目标**：`top: 0`（视口顶部）
- **悬停**：仅 `cursor: pointer`，无其他视觉变化
- **按下（active）**：无视觉变化
- **焦点（focus）**：应保留浏览器默认 focus outline（无障碍要求）

### 8.3 导航栏返回链接

- **目标**：首页URL（`/` 或 `/index.html`）
- **悬停**：无背景色变化、无下划线、无阴影变化，仅 `cursor: pointer`
- **跳转方式**：标准 `<a>` 标签，浏览器默认导航行为

### 8.4 卡片悬停效果

- **当前实现**：卡片无悬停效果
- **可选增强**（非必须）：如需添加悬停反馈，建议使用硬阴影偏移变化：

```css
/* 可选悬停效果（非默认实现） */
.card:hover {
  box-shadow: 4px 4px 0 rgba(0,0,0,1);    /* 阴影偏移量加倍 */
  transform: translate(-2px, -2px);        /* 抵消阴影增加的偏移 */
}
```

> 注意：如果添加悬停效果，必须设置 `transition: none` 或极短的 transition（如 100ms），避免违背布鲁塔主义风格。

### 8.5 无运动策略（No Motion Policy）

本页遵循最小动画原则：

- **禁止使用**：渐入动画、加载动画、卡片翻转、视差效果
- **允许使用**：平滑滚动（smooth scroll）——唯一允许的动画行为
- **无障碍尊重**：应在CSS中添加 `@media (prefers-reduced-motion: reduce)` 媒体查询，禁用平滑滚动：

```css
@media (prefers-reduced-motion: reduce) {
  html {
    scroll-behavior: auto;     /* 禁用平滑滚动 */
  }
  * {
    transition: none !important;
    animation: none !important;
  }
}
```

| 链接类型 | 行为 | 目标 |
|---------|------|------|
| "← 回到首页" | 页面导航（`<a href="/">`) | 首页 |

## 10. 响应式适配

### 10.1 断点定义

| 断点名称 | 宽度范围 | 典型设备 |
|---------|---------|---------|
| Mobile（移动端） | 0px ~ 639px | 手机竖屏 |
| Tablet（平板端） | 640px ~ 1023px | 平板、大屏手机横屏 |
| Desktop（桌面端） | 1024px ~ | 笔记本、台式显示器 |

### 10.2 移动端适配（< 640px）

| 元素 | 适配规则 |
|------|---------|
| 导航栏 | 品牌名保持18px不变；返回按钮文字保持14px不变；padding从16px缩减至12px |

| 页面容器 | 左右padding从16px缩至12px |
| 回到顶部按钮 | 保持40x40不变；bottom从24px缩至16px；right从24px缩至16px |

### 10.3 平板端适配（640px ~ 1023px）

| 元素 | 适配规则 |
|------|---------|
| 导航栏 | 与桌面端一致，无变化 |

| 新闻卡片 | 与桌面端一致，padding保持 `20px 16px` |

### 10.4 桌面端（>= 1024px）

- 所有样式以桌面端为基准（即本文档前述所有规范值）
- 容器在 1200px 后不再扩展，居中显示

## 11. 暗色模式

### 11.1 暗色模式切换

暗色模式通过 `@media (prefers-color-scheme: dark)` 或 `.dark` 类名切换。推荐使用CSS变量实现一键切换。

### 11.2 暗色模式颜色覆盖表

| 令牌 | 亮色值 | 暗色值 | CSS 变量 |
|------|--------|--------|----------|
| 页面背景 | `#c5c9c9` | `#1a1a1a` | `--color-bg` |
| 卡片背景 | `#d8dada` | `#252525` | `--color-card-bg` |
| 导航栏背景 | `#111111` | `#0a0a0a` | `--color-navbar-bg` |
| 主文字色 | `#111111` | `#e5e5e5` | `--color-text` |
| 次文字色 | `#888888` | `#777777` | `--color-text-muted` |
| 白色边框 | `#ffffff` | `#333333` | `--color-border` |
| 纯白 | `#ffffff` | `#e5e5e5` | `--color-white` |
| 纯黑 | `#111111` | `#0a0a0a` | `--color-black` |
| 阴影色 | `rgba(0,0,0,1)` | `rgba(0,0,0,0.8)` | `--color-shadow` |

### 11.3 暗色模式下分类色彩调整

分类颜色在暗色背景下应适当提高亮度，确保可辨识度：

| 分类 | 亮色模式 | 暗色模式（建议） | 调整说明 |
|------|---------|-----------------|---------|
| 研究突破 | `#0040ff` | `#3366ff` | 蓝色亮度提升 |
| 产品发布 | `#1a8a4a` | `#2ecc71` | 绿色亮度提升 |
| 行业动态 | `#b8860b` | `#daa520` | 金色亮度提升 |
| 政策法规 | `#d73333` | `#e74c3c` | 红色亮度提升 |
| 开源项目 | `#6b90ff` | `#85adff` | 浅蓝亮度提升 |

### 11.4 暗色模式实现

```css
@media (prefers-color-scheme: dark) {
  :root {
    --color-bg: #1a1a1a;
    --color-card-bg: #252525;
    --color-navbar-bg: #0a0a0a;
    --color-text: #e5e5e5;
    --color-text-muted: #777777;
    --color-border: #333333;
    --color-white: #e5e5e5;
    --color-black: #0a0a0a;
    --color-shadow: rgba(0,0,0,0.8);
    --color-cat-research: #3366ff;
    --color-cat-product: #2ecc71;
    --color-cat-industry: #daa520;
    --color-cat-policy: #e74c3c;
    --color-cat-opensource: #85adff;
  }

  /* 导航栏边框在暗色模式下 */
  nav {
    border-bottom-color: var(--color-border);
  }

  /* 回到顶部按钮边框 */
  .back-to-top {
    border-color: var(--color-border);
    background-color: var(--color-black);
  }

  .back-to-top svg {
    stroke: var(--color-text);
  }
}
```

### 11.5 暗色模式特殊注意事项

- **硬阴影调整**：暗色模式下阴影颜色从完全不透明（`rgba(0,0,0,1)`）调整为半透明（`rgba(0,0,0,0.8)`），避免阴影在深色背景上过于突兀
- **卡片左侧色条**：继续使用分类色，但采用暗色模式调整后的色值
- **图标容器边框和图标描边**：同样使用暗色模式调整后的分类色值
- **白色边框**：在暗色模式下降为 `#333333`，保持可见但不刺眼

## 13. 与首页的差异对比

### 13.1 总体差异概览

| 维度 | 首页 | 每日新闻页 |
|------|------|-----------|
| 页面类型 | 列表索引页（日期列表） | 内容详情页（当日新闻） |
| 导航栏 | 正向（白色背景，黑色文字/边框） | 反转（黑色背景，白色文字/边框） |
| 时间线 | 有（左侧竖直时间线） | 无 |
| 卡片类型 | 日期概览卡片（小卡片） | 新闻详情卡片（内容更丰富） |
| 分类系统 | 无分类色彩 | 5种分类色彩（左侧色条 + 图标） |
| 图标系统 | 无分类图标 | 5种分类专属SVG图标 |
| 日期排版 | 中等字号 | 超大字号（3rem/48px） |
| 卡片边框 | 统一白色2px边框 | 白色2px + 左侧4px分类色条 |
| 信息密度 | 较低（仅日期+简要摘要） | 较高（图标+标题+分类+时间+描述） |
| 回到首页按钮 | 无 | 有（导航栏右侧） |

### 13.2 导航栏差异详解

**首页导航栏：**
```css
background-color: #ffffff;
border-bottom: 2px solid #111111;
color: #111111;
/* 图标描边: #111111 */
```

**每日新闻页导航栏（反转）：**
```css
background-color: #111111;
border-bottom: 2px solid #ffffff;
color: #ffffff;
/* 图标描边: #ffffff */
/* 新增："← 回到首页" 返回链接按钮 */
```

> 反转逻辑：所有颜色值在首页和详情页之间取反（黑↔白），形成明确的页面层级对比。用户可以通过导航栏的黑白反转直观感知自己已从首页进入了详情页。

### 13.3 卡片内容结构差异

**首页卡片（日期概览）：**
```
┌──────────────────────────────┐
│  2026.06.27                  │  ← 日期标题
│  星期六                       │  ← 星期
│  今日共8条AI新闻              │  ← 新闻数量摘要
└──────────────────────────────┘
```

- 结构简单：仅包含日期、星期、新闻条数
- 无分类色彩
- 无图标
- 无时间标签
- 无描述正文

**每日新闻页卡片（新闻详情）：**
```
┌──────────────────────────────────────────────────────────┐
│▌ ┌──────┐  DeepMind发布新一代多模态推理模型Gemini Ultra  │
│▌ │ ⚗️   │  研究突破 · 08:30                              │
│▌ │      │  Google DeepMind今日发布了Gemini Ultra 3.0...   │
│▌ └──────┘                                                │
└──────────────────────────────────────────────────────────┘
```

- 结构丰富：图标 + 标题 + 分类标签 + 时间 + 描述正文
- 左侧4px分类色条
- 36x36图标容器 + 20x20 SVG图标
- 卡片内使用flex横排布局

### 13.4 日期排版差异

| 属性 | 首页 | 每日新闻页 |
|------|------|-----------|
| 日期字号 | ~1.125rem（18px，作为卡片标题） | 3rem（48px，作为页面级主标题） |
| 字间距 | 0 | -0.03em（紧缩） |
| 行高 | 正常（~1.4） | 紧凑（1.1） |
| 位置 | 卡片内 | 页面顶部独立区域 |
| 副标题 | 无 | "星期六 · 今日AI新闻速递" |
| 视觉重量 | 中等（卡片标题级别） | 极高（页面主标题级别，强烈的视觉冲击） |

### 13.5 卡片列表布局差异

| 属性 | 首页 | 每日新闻页 |
|------|------|-----------|
| 布局方式 | 时间线列表（timeline） | 纯垂直卡片堆叠 |
| 时间线装饰 | 有（左侧竖线 + 节点圆点） | 无 |
| 卡片间距 | 16px | 16px（相同） |
| 卡片边框 | 2px solid #ffffff | 2px solid #ffffff + 4px left {分类色} |
| 卡片阴影 | 2px 2px 0 rgba(0,0,0,1) | 2px 2px 0 rgba(0,0,0,1)（相同） |
