## 1. 页面概述

### 1.1 用途

AI日报首页是该站点的核心入口页面，展示当日AI新闻摘要、历史新闻时间线、站点介绍信息。用户可在此快速了解AI领域每日重要动态，并通过链接跳转至详细新闻页面。

### 1.2 目标受众

- AI领域从业者（研究员、工程师、产品经理）
- 科技行业投资者与分析师
- AI技术爱好者与关注者
- 独立开发者与技术社区成员

### 1.3 内容类型

- **每日新闻摘要**：当日3条分类精选新闻（研究 / 产品 / 行业）
- **历史新闻**：按时间倒序排列的历史新闻条目（时间线形式，默认展示4条）
- **站点信息**：站点简介与作者信息
- **品牌标识**：Logo、站点名称、导航链接

### 1.4 页面结构概览

```
┌──────────────────────────────────────┐
│ 固定导航栏 (Fixed Navbar)            │
├──────────────────────────────────────┤
│ 主内容区 (Main)                       │
│  ├── 今日新闻摘要卡片                 │
│  ├── 历史新闻时间线                   │
│  └── 关于区域（双列卡片网格）         │
├──────────────────────────────────────┤
│ 页脚 (Footer)                         │
├──────────────────────────────────────┤
│ 回到顶部按钮 (Fixed Bottom Right)     │
└──────────────────────────────────────┘
```

## 2. 设计风格定义

### 2.1 视觉风格

- **布鲁塔主义编辑风格（Brutalist Editorial）**：硬朗、直率、拒绝装饰主义
- **等宽字体（Monospace）**：全站使用等宽字体，营造技术感与编辑排版感
- **硬阴影（Hard Shadow）**：无模糊的实色偏移阴影，模拟纸张叠放效果
- **零圆角（Zero Radius）**：所有卡片、按钮、组件均为直角矩形
- **白色边框（White Border）**：2px纯白边框强化元素边界感
- **有限色彩**：以灰阶为基底，仅通过分类标签引入有限色彩

### 2.2 设计基调

- **技术感**：等宽字体 + 暗色文字传达专业、精准的技术气质
- **可信赖感**：编辑排版风格赋予内容权威性与可信度
- **简洁性**：去除一切不必要装饰，让内容本身成为焦点
- **一致性**：全局统一的设计语言，每个组件遵循相同的视觉规则

### 2.3 设计哲学

> "让内容说话。" 页面设计服务于信息传达效率，所有视觉元素（阴影、边框、色彩）均为增强可读性与层次感而存在，而非纯装饰目的。布鲁塔主义风格的选择反映了对内容优先、反过度设计的价值主张。

### 4.6 CJK 等宽字体注意事项

- 全站使用等宽字体，CJK字符（中日韩）宽度为拉丁字符的2倍（全角宽度）
- 日期格式使用 `.` 分隔（如 `2026.06.27`）而非中文习惯的 `/`，以保持等宽视觉一致
- 分类标签使用方括号 `[` `]` 包裹，利用等宽字体对齐特性
- 中英文混排时，标点使用西文标点（`·` 代替中文居中点 `·`）以适应等宽环境
- 字间距 `tracking-tight` 应用于品牌名，缩减中文字符间距

- 层级：`z-50`
