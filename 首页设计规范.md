## 2. 设计风格定义

| 属性 | 说明 |
|------|------|
| **风格名称** | Brutalist Editorial（粗野主义编辑风） |
| **核心理念** | 等宽字体排版、零圆角、硬偏移阴影（hard offset shadow）、灰色调色板、工业/编辑质感 |
| **关键词** | 工业、冷峻、理性、极简、印刷品质感 |

### 设计哲学

- **等宽字体优先**：全站使用 `Geist Mono` 等宽字体栈，所有文本（含中文）均为等宽渲染，传递技术/程序员的严肃感
- **零圆角策略**：所有 `border-radius` 均为 `0px`，拒绝任何柔和的曲线，强调棱角分明的工业美学
- **硬偏移阴影**：使用 `2px 2px 0 rgba(0,0,0,1)` 的纯黑偏移阴影，悬浮时扩大至 `4px 4px`，模拟纸张叠加的物理效果
- **灰阶主调**：背景色 `#c5c9c9`（冷灰）、卡片色 `#d8dada`（浅灰），文字色 `#111111`（近黑），整体呈现低饱和度的印刷质感
- **色彩仅用于语义标记**：彩色仅出现在新闻分类标签（研究/蓝、产品/绿、行业/金）和链接上，主体界面保持无彩色

### 情绪描述

冷峻、理性、专业、克制的工业编辑风格。

### 4.6 CJK 等宽字体渲染说明

- 全站使用 `Geist Mono` 等宽字体栈，中文字符在等宽字体下每个字符占据相同宽度（约 2 个英文字符宽度）
- `<body>` 标签上设置了 `antialiased` 类（`-webkit-font-smoothing: antialiased; -moz-osx-font-smoothing: grayscale`），确保小字号下字体边缘平滑
- 中文字符的 `tracking-tight`（负字间距）在品牌文字上应用，使标题在等宽字体下视觉上更紧凑

## 8. 交互规范

### 8.1 回到顶部按钮

| 属性 | 说明 |
|------|------|
| 滚动阈值 | `window.scrollY > 300` |
| 显示动画 | `opacity` 从 `0` 过渡到 `1`，`transition: opacity 0.2s ease` |
| 隐藏动画 | `opacity` 从 `1` 过渡到 `0`，同时 `pointer-events` 切换为 `none` |
| 滚动行为 | `window.scrollTo({ top: 0, behavior: 'smooth' })`（平滑滚动） |
| 悬浮效果 | `transform: translate(-1px, -1px)`，阴影扩大至 `4px 4px 0 rgba(0,0,0,1)` |
| 过渡时长 | `transform 0.15s ease`, `box-shadow 0.15s ease` |

### 8.2 CTA 按钮

| 属性 | 说明 |
|------|------|
| 悬浮效果 | `transform: translate(-1px, -1px)`，阴影扩大至 `4px 4px 0 rgba(0,0,0,1)` |
| 恢复效果 | `transform: translate(0, 0)`，阴影恢复至 `2px 2px 0 rgba(0,0,0,1)` |
| 过渡时长 | `transform 0.15s ease`, `box-shadow 0.15s ease` |

### 8.3 导航链接

| 属性 | 说明 |
|------|------|
| 悬浮效果 | `text-decoration: underline`（通过 Tailwind `hover:underline`） |
| 过渡时长 | 无显式过渡（浏览器默认） |

### 8.4 动画/运动策略

- 本页面 **不包含** 任何 CSS animation 或关键帧动画
- 所有交互反馈均为 **瞬时状态切换**（hover transform + shadow 偏移）
- 唯一的过渡动画为回到顶部按钮的 `opacity` 渐变（0.2s ease）
- transform 和 box-shadow 的 hover 过渡均为 0.15s ease

### 8.5 减弱运动偏好

- 页面当前未包含 `@media (prefers-reduced-motion: reduce)` 的显式处理
- 建议实现时应在此媒体查询下禁用所有 `transition` 和 `transform` 效果，回到顶部按钮直接切换 `opacity` 无渐变

## 11. 暗色模式

### 11.1 暗色模式切换机制

- 通过 `<html>` 元素的 CSS 类切换实现：`class="light"` ↔ `class="dark"`
- HTML 初始状态为 `class="light"`
- 所有颜色通过 CSS 自定义属性（变量）引用，切换 `.dark` 类后自动更新

### 11.2 暗色模式组件级影响分析

| 组件 | 影响说明 |
|------|---------|
| **页面背景** | `#c5c9c9` → `#0a0a0a`，整体从浅灰变为深黑 |
| **导航栏背景** | `#d8dada` → `#1a1a1a`，卡片色与背景色对比度保持 |
| **导航栏边框** | `2px solid #ffffff` → `2px solid #333333`，白色边框变为深灰 |
| **品牌文字 / 导航链接** | `#111111` → `#ffffff`，文字从深色反转为白色 |
| **Robot Logo** | fill 和 stroke 均跟随 `--color-text-foreground`，自动反转为白色 |
| **今日摘要卡片** | 背景 `#d8dada` → `#1a1a1a`，边框 `#ffffff` → `#333333`，文字 `#111111` → `#ffffff` |
| **分类标签** | 研究 `#0040ff` → `#0040ff`（不变），产品 `#1a8a4a` → `#34d399`（变亮），行业 `#b8860b` → `#fbbf24`（变亮） |
| **CTA 按钮** | 背景 `#111111` → `#ffffff`（反转），文字色 `#ffffff` → `#333333`（反转） |
| **时间线竖线 / 标记** | `#111111` → `#ffffff`，自动反转为白色 |
| **时间线卡片** | 背景 `#d8dada` → `#1a1a1a`，边框 `#ffffff` → `#333333`，文字 `#111111` → `#ffffff` |
| **日期标签** | `#888888` → `#aaaaaa`，弱化文字略微提亮 |
| **"查看详情" 链接** | `#0040ff` → `#0040ff`（不变） |
| **关于卡片** | 与时间线卡片一致：背景、边框、文字均反转 |
| **页脚文字** | `#888888` → `#aaaaaa` |
| **页脚边框** | `#ffffff` → `#333333` |
| **回到顶部按钮** | 背景 `#111111` → `#ffffff`（反转），边框 `#333333`，箭头 SVG stroke 跟随 `--color-border-border` → `#333333` |
| **阴影** | 所有阴影从黑色偏移 `rgba(0,0,0,1)` 变为白色半透明偏移（opacity 0.08~0.2），保留硬偏移风格 |
| **hover 阴影** | inline style 中硬编码 `rgba(0,0,0,1)` → **不受 CSS 变量控制**，暗色模式下 hover 阴影仍为黑色，视觉上可能需要额外适配 |

> **注意**：hover 状态的 `box-shadow: 4px 4px 0 rgba(0,0,0,1)` 在 `onmouseover` 内联事件中硬编码，不引用 CSS 变量，暗色模式下黑色阴影在深色背景上可能不够明显。如需完善暗色模式，应将 hover 阴影也改为 CSS 变量引用。

### Tailwind 配置

页面通过 `<style type="text/tailwindcss">` 中的 `@theme inline` 指令将部分 CSS 变量注册为 Tailwind 主题值：

```css
@theme inline {
  --color-chart-1: var(--color-chart-1);
  --color-chart-2: var(--color-chart-2);
  --color-chart-3: var(--color-chart-3);
  --color-chart-4: var(--color-chart-4);
  --color-chart-5: var(--color-chart-5);
  --color-sidebar-foreground: var(--color-sidebar-foreground);
  --color-sidebar-primary: var(--color-sidebar-primary);
  --color-sidebar-primary-foreground: var(--color-sidebar-primary-foreground);
  --color-sidebar-accent: var(--color-sidebar-accent);
  --color-sidebar-accent-foreground: var(--color-sidebar-accent-foreground);
}
```

同时通过 `@layer base` 设置了全局基础样式：

```css
@layer base {
  body { background: var(--color-background); color: var(--color-text-primary); }
  td, th { @apply break-words; word-break: break-all; word-break: auto-phrase; }
  th { @apply whitespace-nowrap; }
}
```

> 注意：`@layer base` 中引用了 `--color-background` 和 `--color-text-primary`，这两个变量在当前 CSS 变量定义中不存在（实际定义为 `--color-bg-background` 和 `--color-text-foreground`），因此 body 的基础样式实际被 `<body>` 上的 inline style 覆盖。
