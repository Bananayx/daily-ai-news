# daily-ai-news

## 一个由AI撰写的每日AI晚报。

在 TRAE Work 的 自动化 - Code模式 使用以下提示词实现，每日17:00触发：

```text
搜索今日 AI 行业的热点新闻，覆盖以下方面：
1. 重要产品发布或功能更新（如 OpenAI、Google、Anthropic 等公司动态）
2. 融资事件与行业并购
3. 技术突破或重要论文发布
4. 行业标准与规范动态（如 AI 安全框架、数据治理标准的进展）
输出要求：
- 按重要性排序
- 每条新闻包含：标题、详情、信息来源
- 聚焦技术进展和商业动态层面
- 最后附一段 2-3 句的“今日趋势点评”
- 新闻必须具有强时效性，必须是今日的，昨天、前天、以前的一概不要
将index.html作为AI晚报的首页，设计规范参照`首页设计规范.md`，其中顶部导航栏左侧文字写“每日AI晚报”，title改为“每日AI晚报”，关于网站写README.md中的内容，关于作者内容写“懒得写”
将yy-mm-dd.html作为每日AI新闻的内容页，设计规范参照`每日新闻页设计规范.md`，其中顶部导航栏左侧文字写“每日AI晚报”，title改为“yyyy.mm.dd - 每日AI晚报”，页面中每条新闻都要附带与其对应的信息来源的跳转链接，每条新闻不必显示具体时刻（hh:mm）
注意index.html历史新闻部分，你要先查看已有哪些yy-mm-dd.html，时间线上只显示已有的
每次你的任务触发时，都要更新index.html到最新状态，并新增最新的yy-mm-dd.html，旧的不要删，并且直接push到origin/main而不要提交PR
```
