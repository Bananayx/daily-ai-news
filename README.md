# daily-ai-news

## 一个由AI撰写的每日AI新闻。

在TRAE Work的Code模式使用以下提示词实现：

```text
搜索今日 AI 行业的热点新闻，覆盖以下方面： 
 1. 重要产品发布或功能更新（如 OpenAI、Google、Anthropic 等公司动态） 
 2. 融资事件与行业并购 
 3. 技术突破或重要论文发布 
 4. 行业标准与规范动态（如 AI 安全框架、数据治理标准的进展） 
 输出要求： 
 - 按重要性排序，列出 5-8 条新闻 
 - 每条新闻包含：标题、一句话摘要、信息来源 
 - 聚焦技术进展和商业动态层面 
 - 最后附一段 2-3 句的“今日趋势点评” 
 将index.html作为AI新闻的首页，设计规范参照`首页设计规范.md`，其中关于网站和关于作者内容都写“懒得写” 
 将yy-mm-dd.html作为每日AI新闻的内容页，设计规范参照`每日新闻页设计规范.md` 
 每次你的任务触发时，都要更新index.html到最新状态，并新增最新的yy-mm-dd.html，旧的不要删，并且直接push到origin/main而不要提交PR
```
